# Chatbot_Project - Team.올림
크롤링 + AIP를 활용한 서울 청년 부동산 챗봇

## 프로젝트 소개
이 프로젝트를 통하여 청년뿐 아닌 부동산에 대해 지식이 부족한 일반인들이 보다 쉽게 부동산에 대한 정보를 접할 수 있게 되고
더 나은 선택을 할 수 있도록 도움을 주는 것이 목적입니다.
이 프로젝트는 사람들에게 부동산에 대한 기본적인 용어, 계약 처리 및 오류 방지를 할 수 있는 팁을 제공하고, 청년들에게는 참조할 수 있는
정책이과 대출 방향등을 제시하여 부동산에 대한 장벽을 낮춰주는 역할을 기대합니다.
